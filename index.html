What I Learned

// 24 days of building. Zero CS degree. Key insights:

1. Coordination beats raw power.
   8 cheap models in parallel > 1 expensive model serial.

2. Constraints breed creativity.
   1 GB RAM forced architecture that scales to 100 customers.

3. Ship daily, reflect weekly.
   45 ideas in 24 days. Not all perfect, but all shipped.

4. Memory changes everything.
   Agents without memory = colleagues with amnesia.
   Agents with memory = a team that grows.

5. The swarm is the product.
   Single-agent chatbots are commodity.
   Parallel agent swarms are rare.
