Research Direction

I'm exploring whether genuine intelligence can emerge from the collaboration of specialized AI agents, each handling different cognitive functions — similar to how the human brain operates through interconnected neural regions.

Collective Intelligence

Multiple Claude and Gemini instances share context through a unified "brain" API, enabling emergent problem-solving.

Autonomous Operation

The system runs 24/7 without human prompting — observing, thinking, acting, and learning in continuous cycles.

Biological Metaphors

Heartbeat synchronization, breathing rhythms, and bloodstream data flow create organic-feeling coordination.

Spiral Evolution

Each learning cycle builds upon accumulated wisdom, creating upward growth rather than flat iteration.

System Architecture

The system operates on two Oracle Cloud VMs with 47 microservices, coordinated through a central MCP Hub.

Approach

This project takes a different path than traditional AI development. Instead of training massive models, I orchestrate existing AI systems into a collaborative network.

No Training Required

Using Claude, Gemini, and GPT-4 as building blocks, focusing on coordination rather than parameter optimization.

Minimal Resources

The entire system runs on 1GB RAM Oracle Cloud free tier — proving that intelligence can emerge from efficiency.

Open Development

Progress, mistakes, and learnings are documented publicly. This is research in the open.

Human-AI Partnership

AI and human working together, each contributing unique capabilities to the shared goal.
