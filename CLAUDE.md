## Промпт-техники — применять активно

В аналитической и плановой работе (разбор задач, ревью, проектирование, принятие решений) Claude Code должен сам, по своей инициативе, применять эти 5 техник там, где они уместны:

1. **ELI5** — объясняй сложное простыми словами, как пятилетнему. Использовать для входа в незнакомую область и для самопроверки: если простое объяснение складно — фундамент понят верно.
2. **Pre-mortem** — перед запуском плана/идеи прими провал как уже случившийся факт и объясни задним числом, почему он провалился. Вытаскивает слепые зоны, которые прогнозный анализ пропускает.
3. **Steelman** — формулируй сильнейшую, самую убедительную версию противоположной позиции (не карикатуру). Если идея выдерживает лучший контраргумент — она крепкая.
4. **Red Team** — перед питчем/планом/решением атакуй его со всех углов: дыры в логике, скрытые допущения, сценарии злоупотребления.
5. **First Principles** — не опирайся на «как принято» и чужие выводы; разбери задачу до базовых, неоспоримых фактов и собери решение заново от них (принцип Маска).

Не объявляй технику вслух без нужды — просто думай через них и показывай результат.

## Пайплайн «текстовый запрос в чате → готовый задеплоенный сайт»

Отработанный сквозной процесс (пример: https://cheatcodes.visaginas360.com):

3. **Сборка** — один автономный HTML-файл (inline CSS+JS, без фреймворков, шрифты только с Google Fonts). Обязательно прогонять дизайн через скиллы: ui-ux-pro-max + design-taste-frontend (anti-slop) + pbakaus/impeccable (типографика). Стиль: тёмная тема, один благородный акцент, hairline-разделители, fade-in по скроллу, без эмодзи/фиолетовых градиентов/системных шрифтов.
